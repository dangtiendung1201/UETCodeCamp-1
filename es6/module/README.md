# Module

- Import/Export

## Import default

```js
import sum from "./calc.js";
```

## Import target function

```js
import { sum } from "./calc.js";
```

## Import All

```js
import * as calc from "./calc.js";
```

## Export

```js
export default sum;
```

```js
export { sum };
```

```js
export const name = "value";
```
